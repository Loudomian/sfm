title: 闪烁的屏幕
---

如果你发现在你使用 Source Filmmaker 的时候，预览窗口移动镜头疯狂闪烁，这个教程会对你有帮助。

## 产生原因

傻逼 微软 

傻逼 win10

这是 Windows 10 某一次更新产生的 BUG。

## 修复方法
Source Filmmaker 打开后，点击 Windows > Engine Window > Auto Hide Engine Window
